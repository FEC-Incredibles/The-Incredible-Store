## Getting Started

##### Requirements
- Node 6.13.0

##### Run


1. Navigate to the project directory
   ```sh
   cd fec-incredibles
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. To start the project
   ```sh
   npm run build
   npm start
   ```
